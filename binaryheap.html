Diet

Heap is a Purely Functional Binary Heap. Binary Heaps are not common in the functional space, especially because their implementation depends on mutable arrays in order to gain in performance. This functional binary heap is based on Vladimir Kostyukovâ€™s paper and it does support the basic operations on a heap without compromising performance.

Supported Operations

  • add: add a value to the heap
  • remove: remove a value from the heap (it removes the min value)
  • (--) operator: remove a value from the heap (it removes the min value)

  • min: min value in the heap (it does not remove it)

  • toList: list of sorted values in the heap
